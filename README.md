# Web application for selling apartments 
### Own implementation of real world apps

## Assumptions
### The assumptions were to create a web application that would allow:
1. Account registrations
   * Full validation
   * Password hashing ([red] BCrypt)
2. Login 
3. Viewing offers
4. Adding offers
5. Deleting offers
5. Filtering offers
6. Sending e-mails after registration

## Technologies

* JAVA language
* Spring Boot 
  * Spring Security
* Database: H2
* Hibernate
  * Hibernate Validator
- Thymeleaf
- Mail sender
- Passay
